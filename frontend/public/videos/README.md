# Vidéos de background

Placez vos fichiers `.webm` ici.

## Exemples de noms :
- bg-gameplay.webm
- bg-cinematic.webm
- bg-hunters.webm
- bg-arena.webm

## Pour activer les vidéos :

1. Placez vos fichiers .webm dans ce dossier
2. Allez dans `src/components/layout/Layout.tsx`
3. Changez `const [useVideoBackground] = useState(true);`
4. Décommentez la configuration VideoBackground

Voir le guide complet dans `VIDEO_BACKGROUND_GUIDE.md` à la racine du frontend.
