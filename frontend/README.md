# Supervive Frontend

## Stack
- Vite + React 18 + TypeScript
- React Router v6
- TanStack Query (React Query)
- Zustand
- Tailwind CSS + shadcn/ui
- React Hook Form + Zod
- Recharts
- Socket.io-client
- Lucide React (icons)

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Features
- Team management
- Scrim scheduling
- Tournament brackets
- Player/Team stats & leaderboards
- LFT/LFP listings (Mercato)
- Real-time notifications
- Dark mode UI

## Environment Variables
Create `.env` file:
```
VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```
